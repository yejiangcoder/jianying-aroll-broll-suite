# V21 UAT Capsules

This directory is reserved for sanitized production-parity capsules exported by
`tools/export_aroll_v21_uat_capsule.py`.

Capsules may contain only V21 compiler artifacts such as source graph, decision
plan, final timeline, captions, material write plan, validator report, postwrite
report, blocker report, and decision trace.

Do not store `draft_content.json`, `template-2.tmp`, media, images, audio, video,
large runtime JSON, or live Jianying draft folders here.
